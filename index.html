Generative art

by eric

made with OPENRNDR

Generative hatching/cross-hatching. Heavily inspired by @kirbyufo (instagram) and @piterpasma (instagram). First, shapes are generated from some parameters. Next, hashes are added inside the shape's bounding box using circle packing. Next, an OPENRNDR API clips the paths against the shape itself so that only the hashes that are inside the shape are shown. (There are muliple "clipping points" in this image.)
